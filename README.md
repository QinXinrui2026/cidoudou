📚 单词连线 · 多词库

一个智能的单词配对学习工具，通过连线配对的方式帮助记忆单词。支持多词库切换、自定义词库上传、语音朗读等功能。
✨ 功能特点

    🎯 配对学习：左侧显示单词，右侧显示释义，点击配对

    📚 多词库管理：内置新人教八年级下册词库，支持创建多个自定义词库

    📤 词库上传：支持上传 .txt 文件，智能识别多种格式

    🗑️ 词库删除：可删除自定义词库，支持恢复误删的词库

    🔊 语音朗读：支持单个单词朗读和整页朗读

    📊 学习进度：实时显示得分、配对进度、剩余单词

    🔄 自动补全：配对完成后自动补充新词，保持每页10组

    📱 移动端适配：完美支持手机和平板

🚀 快速开始
在线使用

直接打开 index.html 即可使用，无需安装任何依赖。
本地运行
bash

# 克隆或下载项目
git clone <仓库地址>

# 直接用浏览器打开
open index.html

📖 使用指南
1. 选择词库

    点击顶部的下拉菜单选择词库

    切换词库会重置当前学习进度

2. 开始学习

    点击左侧的单词，再点击右侧对应的释义

    配对正确 ✅ 会播放提示音并增加分数

    配对错误 ❌ 会播放错误提示音

3. 上传自定义词库

    点击 📤 上传词库 按钮

    输入词库名称

    选择 .txt 文件（支持多个文件）

    点击 ✅ 智能识别并入库

支持的格式示例：
text

apple, 苹果
book 书本
hello | 你好
school：学校
teacher = 教师

4. 删除词库

    选择要删除的词库

    点击词库旁的 ✕ 按钮

    删除后可点击 ↩️ 恢复词库 找回

5. 其他功能

    🔄 打乱：重新排列当前页面的单词顺序

    ↺ 重置：重置当前词库的学习进度

    🎲 随机10个：从词库中随机抽取10个单词

    🔊 朗读全部：朗读当前页所有未配对的单词

📝 词库格式说明
内置词库

    新人教八年级下册（完整版）

    包含常用单词及中文释义

上传词库格式

每行一个单词配对，支持以下分隔符：

    逗号：apple, 苹果

    Tab：book\t书本

    冒号：school：学校

    竖线：hello | 你好

    等号：teacher = 教师

    括号：apple (苹果)

    破折号：hello - 你好

    空格：student 学生

系统会自动识别英文单词和中文释义的位置。
